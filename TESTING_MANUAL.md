# Blabber — Руководство по ручному тестированию

> **Как пользоваться этим файлом:**
> Каждый пункт — конкретное действие в Telegram с ожидаемым результатом.
> После успешной проверки ставь `[x]` вместо `[ ]`.
> Если что-то пошло не так — пиши заметку под пунктом.

---

## Подготовка к тестированию

### Нужно знать перед стартом

1. **Узнай свой Telegram ID** — напиши [@userinfobot](https://t.me/userinfobot), он ответит числом вида `123456789`.

2. **Убедись, что `.env` настроен.** Найди и раскомментируй строку:
   ```
   ADMIN_TELEGRAM_IDS=ТВОЙ_ID_ЗДЕСЬ
   ```

3. **Перезапусти бота** после изменения `.env`:
   ```bash
   sudo systemctl restart blabber
   sudo systemctl status blabber
   ```
   Статус должен быть `Active: active (running)`.

4. **Для тестов с двумя пользователями** — нужен второй Telegram-аккаунт. Попроси его написать боту `/start` и узнай его Telegram ID.

### Как смотреть логи при ошибке
```bash
sudo journalctl -u blabber -n 50 --no-pager
```

### Проблемы с контекстом (режим Чат)

| Симптом | Возможная причина |
|--------|-------------------|
| Бот не помнит предыдущие реплики | Режим "Вопрос-ответ" — переключи `/mode chat` |
| История очистилась сама | TTL 60 минут — долго не писал, контекст сбросился |
| Кнопки `/reset` не работают | Callback привязан к user_id — другой пользователь не может нажать |
| Ошибка при `/mode` | Проверь миграцию 004: `context_mode` в `users`, таблицы `context_messages`, `context_summary` |

```bash
sqlite3 blabber.db ".schema context_messages"
sqlite3 blabber.db ".schema context_summary"
sqlite3 blabber.db "PRAGMA table_info(users);" | grep context_mode
```

### Проблемы с памятью (профиль и KB)

| Симптом | Возможная причина |
|--------|-------------------|
| `/remember` не работает | Проверь миграцию 005: `user_profiles` |
| `/kb` не работает | Проверь миграцию 006: `kb_documents`, `kb_chunks` |
| Embeddings = NULL при загрузке | `OPENAI_API_KEY` не задан или невалиден — fallback на BM25-only |
| «BM25-only, no API key» при загрузке | Нормально если `OPENAI_API_KEY` не настроен; качество поиска ниже |
| Ошибка при загрузке PDF | Не установлен `pypdf`: `pip install pypdf` |
| Ошибка при загрузке DOCX | Не установлен `python-docx`: `pip install python-docx` |

```bash
sqlite3 blabber.db ".schema user_profiles"
sqlite3 blabber.db ".schema kb_chunks"
sqlite3 blabber.db "PRAGMA table_info(kb_chunks);" | grep embedding
```
